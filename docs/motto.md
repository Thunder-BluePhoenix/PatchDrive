# PatchDrive

**"Keep it moving. Draw to survive."**

---

PatchDrive is a chaotic repair-survival arcade game where the machine never stops and neither do you.

The player is always doing one of three things:

- **Controlling** the machine through terrain, sky, or space
- **Reacting** to disaster before it tears the vehicle apart
- **Drawing** emergency repairs under pressure to keep going

The game never ends. The machine always breaks. The only question is how long you can hold it together.
