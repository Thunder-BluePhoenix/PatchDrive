# Phase 2 — Polish & Feel

**Goal:** Make the game feel alive and reactive. Visuals, feedback, and mobile support.

## Tasks

### Visuals
- [ ] Improve vehicle art so each type looks more distinct
- [ ] Add visible damage effects on the vehicle body
- [ ] Make installed repairs appear visually on the vehicle
- [ ] Add animated disaster effects (storms, earthquakes, meteors)

### Feedback
- [ ] Add sound effects for impacts, repairs, disasters, pickups
- [ ] Add screen shake on hard impacts
- [ ] Add particle effects for damage and explosions

### Terrain
- [ ] Add stronger terrain variety for ground vehicles
- [ ] Add background parallax layers for air and space

### Input
- [ ] Add mobile touch controls for movement
- [ ] Polish the drawing canvas experience on touch screens

### Drawing System
- [ ] Improve shape-scoring model with per-part templates
- [ ] Show post-repair quality rating feedback to the player

## Success Criteria

The game should feel juicy and physical. A first-time player should understand the stakes immediately from audio and visual feedback alone.
