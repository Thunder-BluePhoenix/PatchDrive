# Phase 3 — Content Expansion

**Goal:** Give players more to unlock, more to survive, and more ways to play.

## Tasks

### Vehicles
- [ ] Add more vehicles per category (bike, buggy, jet, orbital craft)
- [ ] Add unlockable vehicle classes gated by garage coins
- [ ] Add vehicle-specific parts list per vehicle

### Upgrades
- [ ] Add ground upgrades: tire grip, suspension, brake strength, torque, chassis strength
- [ ] Add air upgrades: lift, thrust, wing strength, fuel efficiency, turbulence resistance
- [ ] Add space upgrades: thruster power, shield strength, oxygen capacity, reactor efficiency, heat tiles, navigation stability
- [ ] Add universal upgrades: part durability, drawing canvas size, emergency heal efficiency, disaster warning time, coin multiplier

### Disasters
- [ ] Add disaster warning indicators before events hit
- [ ] Add full ground hazard set: earthquake, flood, sandstorm, meteor shower, landslide, ice storm, tornado, mud, rocks
- [ ] Add full air hazard set: thunderstorm, lightning, turbulence, cyclone, volcanic ash, engine icing, air debris
- [ ] Add full space hazard set: asteroid field, solar flare, gravity well, debris storm, radiation burst, oxygen leak, plasma surge

### Missions
- [ ] Add per-run missions with bonus coin rewards
- [ ] Examples: survive 3 disasters, make 5 perfect repairs, travel 5000 distance

### UI
- [ ] Add an in-run pause menu
- [ ] Add a post-run summary screen with stats breakdown

## Success Criteria

A returning player should feel progression and always have something new to unlock or discover.
