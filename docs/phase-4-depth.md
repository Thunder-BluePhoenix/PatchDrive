# Phase 4 — Depth & Replayability

**Goal:** Make every run feel different. Add systems that reward mastery and creativity.

## Tasks

### World Generation
- [ ] Add procedural terrain and world generation
- [ ] Add biome-like zone transitions (desert, ice, city rubble, cloud layer, deep space)

### Boss Events
- [ ] Add boss-like disaster events with multi-phase behavior
- [ ] Examples: mega tornado, colossal asteroid, storm front, satellite collision

### Drawing Recognition
- [ ] Improve drawing scoring with ML-based or template-matching recognition
- [ ] Add part-specific scoring rules (e.g. a wheel must be round, a wing must be flat)
- [ ] Add saved custom parts the player can name and reuse

### Loadouts & Meta
- [ ] Add vehicle loadouts — player picks starting parts
- [ ] Add checkpoint garages mid-run for partial repairs and resupply
- [ ] Add a self-repair system that slowly heals parts at the cost of fuel or coins

### Healing
- [ ] Add repair kits as mid-run collectibles
- [ ] Add emergency heal at cost of run coins
- [ ] Add garage upgrades that improve between-run healing

## Success Criteria

Experienced players should feel the game reward their skill in drawing and driving. No two runs should feel identical.
