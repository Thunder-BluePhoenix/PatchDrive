# Phase 5 — Launch & Community

**Goal:** Ship a polished, shareable game. Build the audience.

## Tasks

### Menus & Settings
- [ ] Add a polished start menu with animation
- [ ] Add settings screen: volume, controls, graphics quality
- [ ] Add an accessible how-to-play screen

### Daily & Social
- [ ] Add daily challenge runs with fixed seeds
- [ ] Add a leaderboard per vehicle type
- [ ] Add shareable score cards (screenshot + stats)

### Packaging
- [ ] Package as a Progressive Web App (PWA) for mobile install
- [ ] Add offline support
- [ ] Add desktop build via Electron or Tauri (optional)

### Monetization (Optional)
- [ ] Cosmetic vehicle skins
- [ ] Cosmetic repair drawing themes
- [ ] One-time unlock packs — no pay-to-win

### Release Targets
- [ ] itch.io release
- [ ] Newgrounds release
- [ ] Mobile app stores (if PWA proves stable)

## Success Criteria

A stranger who finds the game can pick it up, understand it in under 60 seconds, share their score, and come back the next day.
